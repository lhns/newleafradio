New Leaf Radio

Start listening now!

Use the controls provided below to start listening to our little public radio! For dynamic weather support, make sure that you are here on a secure connection (https instead of just http). Setting this is easy-as-pie and all you have to do is make sure your url looks like this: https://newleafradio.glitch.me/radio

This is a young project and we want to make it the best we can. Dynamic weather can be tricky and all the controls are more or less from scratch. If you see something that isn't working right let me know. As for how to contact me, I'll take direct messages over Reddit at Terrible_At_Games. I'll be updating the contact info later on so that the act of sending critique is as easy as possible.
